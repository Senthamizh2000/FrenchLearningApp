# 🎨 French Learning App  

French Learning App is an Android application designed to help users learn French color names with audio pronunciation.  

## 📱 Features  
✔️ Play audio pronunciation for different colors  
✔️ Simple and user-friendly UI  
✔️ Works on both emulators and real devices  

## 🛠️ Tech Stack  
- **Language:** Java  
- **Framework:** Android SDK  
- **UI:** ConstraintLayout  
- **Audio:** MediaPlayer API
